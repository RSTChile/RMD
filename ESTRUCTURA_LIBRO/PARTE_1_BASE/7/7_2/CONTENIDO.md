7.2 Condición previa: CSS
Antes de cualquier cálculo del ICSGS, el analista debe verificar la existencia de Condición de Sincronización
Sistémica (CSS):
•
•

Si CSS = 0 → No existe colapso sistémico sincronizado → ICSGS = 0
Si CSS > 0 → Existe sincronización de daño → Procede cálculo del ICSGS

Definición CSS:
Existencia verificable de afectación simultánea o secuencial acoplada de dos o más nodos críticos
interdependientes con capacidad de propagación sistémica.
