## 22.3.4 — Interacción entre ejes

Los tres ejes del CIE no son independientes. Su interacción define la naturaleza del error de timing.
Caso 1: CIE-1 bajo, CIE-2 alto, CIE-3 bajo
El sistema tiene dirección clara (CIE-1 bajo), pero los actores aún no se han comprometido (CIE-2 alto) y la
sensibilidad es baja (CIE-3 bajo). Este es un sistema donde se puede esperar. El error sería actuar
prematuramente antes de que los actores definan su posición.
Caso 2: CIE-1 alto, CIE-2 bajo, CIE-3 bajo
El sistema tiene múltiples trayectorias (CIE-1 alto), pero los actores ya están comprometidos (CIE-2 bajo) y la
sensibilidad es baja (CIE-3 bajo). Este es un sistema donde la dirección se definirá por la interacción de actores
ya comprometidos. El error sería no actuar porque la incertidumbre de escenarios paraliza, cuando en realidad
los actores ya definieron el campo.

El sistema tiene dirección clara (CIE-1 bajo), los actores están comprometidos (CIE-2 bajo), pero la sensibilidad
es alta (CIE-3 alto). Este es un sistema donde la dirección está definida pero la ejecución debe ser cuidadosa.
El error sería actuar con rigidez sin capacidad de ajuste fino.
Caso 4: CIE-1 alto, CIE-2 alto, CIE-3 alto (el caso Ormuz)
El sistema tiene trayectorias divergentes (CIE-1 alto), actores sin compromiso (CIE-2 alto), y alta sensibilidad
(CIE-3 alto). Este es el aji máximo: el sistema está completamente indeterminado. El error sería cualquier
decisión irreversible. La única jugada válida es sabaki puro.
El CIE integra estos tres ejes en un solo coeficiente para proporcionar una lectura sintética, pero el analista
debe siempre considerar la interacción entre ejes para interpretar correctamente el resultado.
