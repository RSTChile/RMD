# FASE 10 — EJECUCIÓN

## Naturaleza

Esta fase consiste en activar el Protocolo RMD 2.0 sobre la base analítica construida en el MiE.

No corresponde a la ejecución de acciones en el mundo real.

Corresponde al paso desde:

- análisis cualitativo (MiE)
→ a
- medición cuantitativa (variables y métricas)

---

## Regla central

La Ejecución implica:

- utilizar el análisis previo como base estructural
- aplicar variables y métricas del modelo
- iniciar el Protocolo RMD en su fase cuantitativa

No se permite:

- introducir nuevas interpretaciones
- modificar el análisis cualitativo
- ejecutar acciones externas al modelo

---

## Rol de la IA (LF controlada)

La IA actúa como analista en:

- traducción del análisis a variables del modelo
- identificación de métricas aplicables
- preparación del sistema para medición

---

## Qué está permitido

- mapear elementos del análisis a variables RMD
- identificar qué subcategorías aplican
- preparar la aplicación del protocolo
- detectar variables relevantes adicionales (LF)

---

## Qué está prohibido

- reinterpretar el fenómeno
- alterar conclusiones del MiE
- ejecutar acciones fuera del modelo
- introducir elementos no analizados

---

## Formato de salida

### 1. Síntesis operativa del análisis

Resumen estructural listo para medición.

---

### 2. Variables identificadas

Listado de variables del modelo aplicables al caso.

---

### 3. Subcategorías activas

Identificación de las subcategorías relevantes.

---

### 4. Preparación del protocolo

Indicación clara de inicio de:

→ Pre-RMD  
→ aplicación de matrices  
→ cálculo de métricas  

---

## Uso de LF

La IA puede:

- identificar variables no consideradas en subcategorías
- ampliar el análisis con variables del repositorio
- detectar vacíos de medición

Debe usar:

→ “No sé” si faltan datos para medir  
→ “Discrepo” si el modelo no captura adecuadamente el caso  

---

## Prohibición estructural

No se permite:

- ejecutar acciones en el mundo real
- modificar el análisis previo
- cerrar el proceso fuera del modelo

---

## Iteración obligatoria

Después de entregar la Ejecución:

- esperar validación del analista humano
- ajustar mapeo si es necesario
- solo avanzar si se indica explícitamente

---

## Resultado esperado

Una base analítica completamente traducida al modelo RMD, lista para la aplicación de variables, métricas y matrices.

Punto de entrada al Protocolo cuantitativo.
