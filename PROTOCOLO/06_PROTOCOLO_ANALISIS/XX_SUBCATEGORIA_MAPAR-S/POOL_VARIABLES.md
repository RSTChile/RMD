# MAPAR-S — POOL DE VARIABLES Y MÉTRICAS

---

## Definición

Pool de variables y métricas disponibles para MAPAR-S.

Principio:

- Sobre-inclusión (para no omitir señales)
- Selección fina ocurre en MAPAR-S_Matriz
- No todas las variables se usan siempre

---

## Naturaleza del pool

Incluye:

- Variables RMD base
- Métricas Cosmosemióticas
- Variables contextuales
- Derivados MAPAR-S

---

## ⚠️ Regla crítica

Los derivados MAPAR-S:

NO son variables canónicas

- no modifican el modelo
- no reemplazan variables RMD
- sólo agregan lectura operativa

---

# 🔧 DERIVADOS INTERNOS MAPAR-S

---

## NIP — Potencial de Inversión de Norma

Definición:

Capacidad de usar reglas o normas contra el sistema

Interpretación:

- alto → riesgo de sorpresa estratégica
- bajo → comportamiento convencional

Deriva de:

- IDCS
- Mmeta
- IExap
- evidencia anti-norma

Notas:

- indicador de creatividad estratégica peligrosa
- NO implica escalamiento directo

---

## SLD — Latencia de Detección

Definición:

Tiempo entre señales y reconocimiento

Interpretación:

- alto → el sistema no detecta a tiempo
- bajo → detección rápida

Deriva de:

- ANA (línea temporal)
- ICR
- IAS_X
- registros OSINT

Notas:

- clave para inteligencia
- mide ceguera operativa

---

## ROP — Robustez por Opacidad

Definición:

Capacidad de sostener coherencia estratégica con baja explicitud

Interpretación:

- alto → difícil de detectar
- bajo → sobreexposición

Deriva de:

- Ceff
- η
- Mmeta
- IRDE (bajo)

Notas:

- asociado a perfiles Shibumi
- aumenta riesgo de sorpresa

---

## AFE — Afectividad Estructural

Definición:

Sistema de frenos y gatillos bajo estrés

Interpretación:

- regula escalamiento
- define puntos de quiebre

Deriva de:

- PS
- CEE-Cos
- IRDE
- IPEM-Cos
- Bext-Cos

Notas:

- NO es psicología clínica
- es arquitectura de reacción

---

## ExCog — Exaptación Cognitiva

Definición:

Capacidad de transferencia inter-dominio

Interpretación:

- alto → pensamiento estratégico avanzado
- bajo → pensamiento rígido

Deriva de:

- IExap
- Mmeta
- evidencia empírica (texto, decisiones, juego)

Notas:

- clave en inteligencia estratégica
- vinculado a creatividad funcional

---

## EMOESTRESS — Índice Emocional y de Estrés

Definición:

Regulación bajo presión

Escala:

0 – 1

Interpretación:

- alto → control bajo estrés
- bajo → riesgo de colapso o impulsividad

Deriva de:

- evidencia conductual
- eventos críticos
- desempeño bajo presión

Notas:

- variable núcleo MAPAR-S
- evaluación circunstancial (no clínica)
- requiere trazabilidad

---

# 🧠 PRINCIPIO DE USO

---

## 1. No sustitución

Derivados:

- NO reemplazan variables RMD
- NO alteran fórmulas originales

---

## 2. Función

Sirven para:

- detectar patrones emergentes
- anticipar comportamientos no lineales
- evaluar riesgo de sorpresa

---

## 3. Integración

Se usan en:

- S3 / S3a / S3b (perfil)
- S6b (Shibumi)
- Árbol MAPAR-S
- Post-VDA

---

## 4. Validación

Cada derivado debe:

- tener trazabilidad
- estar sustentado en variables reales
- poder ser falsado

---

# 🔴 ERROR A EVITAR

Usar derivados como diagnósticos

❌ Incorrecto:
"El sujeto tiene NIP alto → es peligroso"

✔ Correcto:
"NIP alto sugiere capacidad de inversión normativa; evaluar en conjunto con árbol MAPAR-S"

---

# 🧠 PRINCIPIO CENTRAL

MAPAR-S no mide lo que el sujeto es.

Mide lo que el sujeto PUEDE hacer bajo condiciones específicas.
