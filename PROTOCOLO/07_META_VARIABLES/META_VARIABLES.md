# META VARIABLES — RMD 2.0

## DEFINICIÓN

Las metavariables son el nivel superior del modelo.

No describen partes del sistema.

Describen:
- su estado global
- su dinámica
- su proyección
- su coherencia

---

## FUNCIÓN

Las metavariables cumplen 4 funciones:

1. Integrar variables y métricas  
2. Evaluar riesgo sistémico  
3. Proyectar escenarios  
4. Validar el análisis (anti-sesgo)  

---

## CLASIFICACIÓN

### 1. META-ANÁLISIS

- VDA → disenso estructural  
- ANA → base histórica  
- IGRMD → estado global del sistema  

---

### 2. META-RIESGO

- Bext(t) → riesgo de colapso  
- CEE(t) → percepción de colapso  
- IPEM → probabilidad de escalada  
- IIED → impacto estratégico dinámico  

---

### 3. META-COSMOSEMIÓTICA

- Bext-Cos → colapso narrativo  
- CEE-Cos → percepción semiótica  
- IPEM-Cos → escalada narrativa  
- VDA-Cos → disenso semiótico  
- IDSΨ → desalmamiento  

---

### 4. META-AJUSTE

- MACC → ajuste climático sistémico  

---

## ORDEN EN EL PIPELINE

Las metavariables se aplican en:

→ POST análisis de subcategorías  
→ PRE cierre cosmosemiótico  
→ PRE resumen ejecutivo  

---

## REGLAS DE USO

1. No reemplazan variables base  
2. No se interpretan aisladas  
3. No se usan como conclusión directa  
4. Requieren contexto del sistema completo  

---

## ADVERTENCIA

Una metavariable sin contexto:

→ es un número vacío  

Una metavariable bien interpretada:

→ es una decisión estratégica
